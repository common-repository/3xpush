The plugin allows you to join push subscriptions to the site, as well as to send out notifications when creating a post. An additional opportunity to earn money on subscriptions.

== Description ==

Install this plugin and connect your visitors for your update notifications for one click!

This is really easy:

1. Your visitors subscribes for push sending
2. You create a new post
3. Choose who will receive your push notifications
4. Click Save and done!

All you need just sign up on 3xpush.com, create api key and add your site, copy api key and site id on plugin Settings page and save.

In addition to newsletters, you will earn on your subscribers! Our other users will send their advertising, and you will receive your commission from this. You can also sell your subscribers on the exchange, if necessary, or buy if you want to increase your audience for push sending! Triple profit from push subscriptions along with 3xpush!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/3xpush` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->3XPUSH screen to configure the plugin
4. Sign up on 3xpush.com
5. Go to Settings page, on API tab create your an api key, the copy the key
6. Paste your key into the field API KEY, the save
7. Go to Sites page, add your site
8. Copy site id near #
9. Paste the site id into Site ID filed, then save
10. Done! Also you can specify additional options for subscribing after this


== Frequently Asked Questions ==

= Is this free push sending? =

Absolutely! Sending for your subscribers is fully free for you! But our other users may send their ad by your subscribers and pay money you for this

= If i don't want that someone will send ads for my subscribers? =

No problem! You can switch off monetization in your 3xpush.com account on Settings page, after this your subscribers never will get any ads from us.

= How often i can do push sending? =

You can do it how often how you want, but we have limit 1 unique notification per day

= Is this plugin fit for HTTP site? =

Yes, but only sending notifications for your subscribers with API KEY, subscribing on your site will be unavailable

= Which conditions i can use for sending? =

You can use conditions by: countries, sites, languages, tags, and specify sending time
